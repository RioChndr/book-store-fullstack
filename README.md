# Readme

**Book Store** website.

## Techstack

- ExpressJs
- Nextjs
- Prisma.io
- Docker

## Feature 

- Login
- Register
- List Books
- Detail books

## Install

Backend : 

```
$ docker compose run backend-next sh -c "yarn install"
$ docker compose up -d
```

Frontend :

```
$ npm install
$ npm run dev
```
